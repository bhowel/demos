Basic Geometrical Elements

Point

A single location in space. To draw a point, use a small, solid circle or dot. Use a capital letter to name a point. For example, point A.

Line

Set of points that continue infinitely in one dimension. To draw a line, place an arrow on each end to show that the line goes on without end.

Lines contain an infinite number of points. You can specify one section of a particular line by drawing and naming two points. The following line can be called line AB or line BA.

Line Segment

Part of a line defined by two points called endpoints. To draw a segment, place endpoints at each end of a line.

Ray

Set of points that starts at one endpoint and continues infinitely in one direction. To draw a ray, extend a line out from an endpoint and place an arrow at the end.

Angle

Two rays that start from the same endpoint. To draw an angle, extend two rays from the same endpoint.

Plane

A flat surface with an infinite extent. To draw a plane, create a box and place arrows around it to show that it extends in all directions without limit.

The technical definition of plane is a surface that contains all the straight lines that connect any two points within the plane.

 

Parallel Lines

Two lines that never cross or meet.

Intersecting Lines

When two lines cross or meet at some point, they are intersecting lines.

 

Vertex

The point where two rays cross or meet. In the following angle, point A is the vertex.
