Installing TuffDataList

You can download TuffDataList from here. You can also download the seed application, which contains the TuffDataList code.

After you download the file, unzip the file to any location. You'll get a folder named tuffDataList.

Put the tuffDataList folder anywhere under the top-level folder of your web application. For example, if your application is in the MyApp folder, you can put tuffDataList right under MyApp:
