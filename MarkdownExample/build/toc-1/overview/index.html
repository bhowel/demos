Before you start

Before you start using TuffDataList, go through GETTING STARTED to install and set up the plugin.

In GETTING STARTED, we showed you how to use the setupLists function in the .js file for your page to initialize TuffDataList. We're now going to show you how to use TuffDataList to create lists. You can put your file together any way you like, as long as the list creation runs after the setupLists function.
